WORLDTEXT WEBSITE

Open index.html to play all eleven chapters from one clock.
Open chapter-01.html through chapter-11.html for standalone chapter pages.

GitHub Pages:
1. Upload this entire folder without changing filenames.
2. Make index.html the site entry point.
3. No build step or external libraries are required.

Controls:
SPACE play/pause
LEFT/RIGHT seek one second
SHIFT+LEFT/RIGHT previous/next chapter
V voiceover
M procedural score
F cinema mode
ESC leave cinema mode

The browser Speech Synthesis API performs the voiceover written into each twin.json.
The SCORE control creates a minimal procedural cue from Web Audio; no sound files are required.
