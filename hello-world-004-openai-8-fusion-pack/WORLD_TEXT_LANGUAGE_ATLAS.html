Dialect Conflicts

where integration breaks
geometry vs graphMeshes exist but meaning is lost.

Every mesh-producing action must create graph records.

decor vs rulesWorlds look different but play the same.

Every family carries rule and zone payload.

SAY vs BOTSTwo command languages for actors.

Both produce <DirectiveDraft>.

menu vs registryGames become special cases.

Every mode is a <GamePlugin>.

room vs eventMultiplayer duplicates feature logic.

Room sync mirrors committed events.

save vs commitButtons imply inconsistent persistence.

Commit applies; save serializes; deploy instantiates.
