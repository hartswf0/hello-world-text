How To Use

Chunk By Chunk

Open the target and one reference side by side. For each feature, run the manual test, mark what passes, then implement only the next missing substrate. Do not merge a surface until its substrate contract exists.

The goal is not feature volume. The goal is making every visible control change durable state that another mode, film take, or Paris case can consume.
