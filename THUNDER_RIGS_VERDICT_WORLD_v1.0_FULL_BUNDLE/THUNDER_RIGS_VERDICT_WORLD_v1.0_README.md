# THUNDER RIGS · VERDICT WORLD v1.0

## Start
Serve this folder through HTTP or HTTPS, then open `THUNDER_RIGS_VERDICT_WORLD_v1.0_FULL.html`.

```bash
python3 -m http.server 8000
```

## Play
- Drive: WASD, arrows, or the left touch control.
- Actions: Jump, Brake, Boost.
- Morph: quick material strip or LAB → RIG.
- Body shortcuts: 1 rigid, 2 elastic, 3 ragdoll, 4 jelly.

## Verdict game
1. CASE → choose a scene and record a 12-second incident.
2. Party A and Party B watch and seal separate witness accounts.
3. Each party tunes a reconstruction and declares claims.
4. Each party challenges the opponent's claims.
5. The judge scores path, physics, identity, provenance, and challenge accuracy.
6. Export the sealed source record and adjudication stream as one case file.

## Interface
- SOUND and LAB live in the unified top bar and never float over it.
- The Lab is tabbed into Rig, Scene, Audio, and System.
- The bottom case deck collapses while an incident is recording.
- Evidence views remain on the outer edge: Source, A, B, Overlay.
